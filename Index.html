Welcome to EventMaster

Plan events, track assignment deadlines, set reminders, and stay organized. This documentation walks you through common tasks.

Quick start
  1. Create a project or class
  2. Click "New event" to add a deadline
  3. Set recurrence & reminders
  4. Invite collaborators or enable calendar sync

Creating events

Events are the core unit in EventMaster. An event has: title, date/time, type (event or assignment), optional notes, and reminders.

Example: Add a new assignment

// Steps in the app (UI)
1. Click "New" → "Assignment"
2. Title: "Final essay"
3. Due: 2026-05-12 23:59
4. Reminders: 3 days before, 12 hours before
5. Save

Tip: Use the "assignment" type to show grading fields and attachrubrics (when available).

Reminders & notifications

Choose one or more reminder rules. EventMaster supports push notifications, email summaries, and SMS (where available).

FAQ

How do I change time zone?

Open Settings → Account → Time zone. Event times will be shown in your selected zone; collaborators will see events in their own settings unless "force timezone" is enabled.

Can I export all my events?

Yes. Go to Settings → Data → Export and choose CSV or .ics. CSV includes custom fields and tags.

How do I get reminders via SMS?

SMS is available in supported regions under Settings → Notifications. You'll need to verify your phone number.

Developer / API notes (optional)

EventMaster offers a REST API for reading and creating events. Use OAuth2 for authentication. Example request:

GET /api/v1/events
Authorization: Bearer <token>
Response: 200 OK
[ { id: 1, title: "Final essay", due_at: "2026-05-12T23:59:00Z" }, ... ]
